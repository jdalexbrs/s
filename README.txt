GodestMC - Static Expanded Package

Archivos incluidos:
- index.html, about.html, faq.html, contact.html, tos.html, privacy.html
- script.js (comportamiento: router, toasts, copiar IP)
- README.txt (este archivo)

Cómo usar:
1. Descomprime en tu hosting o súbelo a WordPress (p.ej. en una página usando un iframe, o colocando los archivos en la carpeta del tema).
2. index.html es la página principal. Todas las páginas usan script.js para comportamiento.
3. Para activar la API de jugadores online, edita script.js y habilita la función correspondiente (comentada por defecto).

Si necesitas que migre más contenido exactamente como en los .jsx, dime qué página quieres que pulienda y la convertiré con mayor fidelidad.
